-->> Student Attendance Risk Predictor
A Machine Learning project that predicts whether a student is at risk of detention based on attendance data.

-->> Features
1) 75% Attendance Requirement Calculator
2) Risk Prediction using Random Forest Classifier
3) Interactive Streamlit Web App
4) Model Accuracy: ~98%

-->> Tech Stack
1) Python
2) Pandas
3) Scikit-learn
4) Streamlit
5) Joblib

-->> Model Performance
1) Accuracy: 98%
2) Precision & Recall balanced
3) Trained on 500+ synthetic records

-->> How to Run

pip install -r requirements.txt
py generate_dataset.py
py train_model.py
py -m streamlit run app/app.py
